Markup Markup customization and highlights

Mitchell Hayes - is a minimal, creative and beautiful HTML5 template that is suitable for artists, photographers and other specialists with informative portfolio.

Mitchell Hayes was built with the latest Bootstrap 4 framework, so that this template will look great on any device.

General Markup

The general template structure is the same throughout the template. Here is the general structure.

<!DOCTYPE html>
<html lang="en">

<head>
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="MobileOptimized" content="320">
<meta name="HandheldFriendly" content="True">
<meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=no">
<title>Mitchell Hayes</title>
<meta name="description" content="Mitchell Hayes — Portfolio HTML5 Template">
<!-- Disable tap highlight on IE-->
<meta name="msapplication-tap-highlight" content="no">
<meta name="msapplication-TileImage" content="assets/images/touch/ms-touch-icon-144x144-precomposed.png">
<!-- Add to homescreen for Chrome on Android-->
<meta name="mobile-web-app-capable" content="yes">
<meta name="application-name" content="Mitchell Hayes — Portfolio HTML5 Template">
<link rel="icon" sizes="192x192" href="assets/images/touch/chrome-touch-icon-192x192.png">
<!-- Add to homescreen for Safari on iOS-->
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-title" content="Mitchell Hayes">
<link rel="apple-touch-icon" href="assets/images/touch/apple-touch-icon.png">

<link href="https://fonts.googleapis.com/css?family=Montserrat:300,400,500,600,700%7CPlayfair+Display:400,700" rel="stylesheet">

<!-- build:css assets/css/style.css-->
<link href="assets/css/bootstrap.min.css" rel="stylesheet">
<link href="assets/css/font-awesome.min.css" rel="stylesheet">
<link href="assets/css/photoswipe.css" rel="stylesheet">
<link href="assets/css/photoswipe-default.css" rel="stylesheet">
<link href="assets/css/swiper.min.css" rel="stylesheet">
<link href="assets/css/animate.min.css" rel="stylesheet">
<link href="assets/css/style.css" rel="stylesheet">
<!-- endbuild-->
</head>
